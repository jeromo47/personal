# Matriz de gastos prorrateados 2025 - Menéndez Pidal v2

## Fórmula de trabajo
**Gasto imputado habitación = gasto total x coeficiente temporal x coeficiente habitación**

Donde:
- **coeficiente temporal** = días ocupados o afectos / días del periodo
- **coeficiente habitación** = superficie habitación + reparto proporcional de zonas comunes / superficie total vivienda

## Base de superficies usada
- superficie total vivienda: **82 m²**
- H1 privada: `5,88 m²`
- H2 privada: `6,16 m²`
- H3 privada: `6,16 m²`
- H4 privada: `9,80 m²`
- zonas comunes a repartir: `82 - 28 = 54 m²`
- reparto de comunes por habitación: `13,50 m²`

## Coeficientes provisionales de trabajo
- H1: `(5,88 + 13,50) / 82 = 23,63 %`
- H2: `(6,16 + 13,50) / 82 = 23,98 %`
- H3: `(6,16 + 13,50) / 82 = 23,98 %`
- H4: `(9,80 + 13,50) / 82 = 28,41 %`

## Criterios adicionales

- gastos generales de estructura: reparto por **coeficiente habitación**
