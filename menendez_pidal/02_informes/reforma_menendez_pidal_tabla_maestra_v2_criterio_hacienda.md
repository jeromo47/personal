# Reforma Menéndez Pidal tabla maestra v2 criterio Hacienda

## Objetivo
Cruzar la tabla maestra económica de reforma con la solidez documental ya revisada para dejar una versión más útil de cara a Hacienda.

## Criterio de esta v2
Esta versión ya distingue cuatro niveles prácticos:
- **Sí, fuerte**
- **Sí, provisional**
- **Pendiente de soporte / criterio**
- **No usar**

No pretende cerrar todavía la calificación técnico-fiscal exacta de cada gasto como mejora, reparación, mobiliario o equipamiento, pero sí deja muy encaminado:
- qué está bien soportado
- qué está solo pagado pero no facturado fino
- qué conviene dejar fuera

## Conclusión v2
La tabla ya deja tres niveles bastante claros:
1. **obra principal muy bien trazada pero pendiente de cierre fiscal/documental final del contratista**
2. **bloque fuerte adicional de equipamiento/material ya muy usable**
3. **capa floja/dudosa separada para no contaminar el expediente**

Esto ya permite trabajar con una cifra de reforma más afinada sin mezclar todo indiscriminadamente.
